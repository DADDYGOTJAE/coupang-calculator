채널 링크 편집

변경하면 GitHub에 자동 커밋되고, 1~2분 후 사이트에 반영됩니다.

1단계 · GitHub 토큰 등록 (최초 1회)

이 페이지가 GitHub에 변경사항을 저장하려면 토큰이 필요해요. 토큰은 이 브라우저에만 저장되고, 외부로 전송되지 않습니다.

토큰 받는 방법 (3분)
  1. github.com/settings/personal-access-tokens/new 접속
  2. Token name: coupang-calculator-admin
  3. Expiration: 90 days (만료되면 새로 발급)
  4. Repository access: "Only select repositories" → coupang-calculator 선택
  5. Permissions → Repository permissions → Contents: Read and write
  6. Generate token 클릭 → 노란 박스의 github_pat_... 토큰 복사
  7. 이 페이지로 돌아와서 위 입력창에 붙여넣기 → 토큰 저장

2단계 · 채널 링크 편집

아이콘(이모지)·이름·URL을 입력하세요. 빈 행은 자동으로 제외됩니다.
